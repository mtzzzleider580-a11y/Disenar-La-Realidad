Diseñar la Realidad
Manifiesto Digital · Diseño Estratégico · Marca Personal

Overview
Diseñar la Realidad es un proyecto web editorial desarrollado en HTML5 y CSS3 que explora el diseño como sistema estratégico aplicado a la disciplina, la identidad y la construcción de resultados sostenibles.
El proyecto combina:
Diseño minimalista de alto contraste
Layout tipo revista digital
Arquitectura visual basada en grid
Narrativa estructurada como manifiesto
Este repositorio representa una pieza conceptual enfocada en estructura, claridad visual y coherencia estratégica.

Tech Stack
HTML5
CSS3
CSS Variables (:root)
CSS Grid
Flexbox
Google Fonts (Playfair Display, Inter)
Responsive Design (Media Queries)

Architecture & Design Decisions

Layout System
Grid de 6 columnas
Distribución asimétrica editorial
Sidebar sticky estratégica
Hero full-screen con overlay

Design System
Variables CSS para escalabilidad
Separación clara de responsabilidades (HTML estructura / CSS presentación)

Paleta minimalista:
Fondo oscuro dominante
Texto claro de alto contraste
Color acento rojo estructural

Experiencia Visual
Línea vertical fija como recurso editorial
Imágenes con efecto hover (scale transform)
Jerarquía tipográfica fuerte en hero
Espaciado amplio para lectura cómoda

Responsive Behavior
Adaptación automática para dispositivos < 900px:
Layout pasa a una sola columna
Sidebar deja de ser sticky
Se elimina la línea editorial fija
Ajuste de padding y proporciones
