# Diseñar la Realidad

## 1. Descripción del proyecto

**Diseñar la Realidad** es una landing page desarrollada con **HTML5** y **CSS3** que presenta un manifiesto digital sobre diseño estratégico, identidad y construcción de resultados sostenibles. El proyecto adopta un enfoque editorial, combinando una estética minimalista con una estructura visual basada en cuadrículas para ofrecer una experiencia de lectura clara y atractiva.

Su propósito es demostrar buenas prácticas en el desarrollo frontend mediante el uso de una estructura semántica, un sistema de diseño escalable y una interfaz adaptable a diferentes tamaños de pantalla.

---

## 2. Tecnologías

* **Lenguaje de marcado:** HTML5
* **Estilos:** CSS3
* **Variables de estilo:** CSS Variables (`:root`)
* **Sistema de diseño:** CSS Grid
* **Maquetación flexible:** Flexbox
* **Tipografías:** Google Fonts (Playfair Display e Inter)
* **Diseño adaptable:** Responsive Design mediante Media Queries

---

## 3. Arquitectura

El proyecto sigue una **arquitectura basada en la separación de responsabilidades**, donde la estructura del contenido y la presentación visual se encuentran desacopladas.

* **HTML**

  * Define la estructura semántica de la página y organiza las diferentes secciones del contenido.

* **CSS**

  * Centraliza la presentación visual, incluyendo el sistema de colores, tipografía, distribución mediante Grid y Flexbox, animaciones y comportamiento responsive.

Esta organización facilita el mantenimiento, la reutilización de estilos y la escalabilidad del proyecto.

---

## 4. Funcionalidades

El proyecto incorpora las siguientes características:

* Landing page con diseño editorial.
* Hero principal de pantalla completa.
* Diseño minimalista de alto contraste.
* Sistema de maquetación mediante CSS Grid.
* Uso de Flexbox para distribución de componentes.
* Sidebar con comportamiento *sticky* en pantallas grandes.
* Sistema de variables CSS para facilitar el mantenimiento del diseño.
* Línea editorial fija como elemento gráfico.
* Imágenes con efecto *hover* mediante transformaciones CSS.
* Jerarquía tipográfica basada en fuentes serif y sans-serif.
* Adaptación responsive para dispositivos móviles y tablets.
* Distribución automática del contenido en una sola columna en pantallas pequeñas.

## 5. Cómo ejecutarlo

### Requisitos

* Navegador web moderno (Google Chrome, Microsoft Edge, Mozilla Firefox o similar).

### Pasos

1. Clonar el repositorio.

```bash
git clone <URL_DEL_REPOSITORIO>
```

2. Acceder al directorio del proyecto.

```bash
cd Disenar-La-Realidad
```

3. Abrir el archivo principal.

```text
index.html
```

También puede ejecutarse utilizando la extensión **Live Server** de Visual Studio Code para visualizar los cambios en tiempo real durante el desarrollo.
